## Installation

1. **Clone the repository**:
   ```sh
   git clone https://github.com/your-username/structure-playwrite.git
   cd gpt-structure-playwrite
Install dependencies:

npm install
Usage
Running Tests
Run all tests:


npm test
Run tests with UI:


npm run test\:ui
Run tests in debug mode:


npm run test\:debug
Generate and open Allure report:


npm run test\:report
Record tests:


npm run test\:record
Linting and Formatting
Lint the code:


npm run lint
Format the code:


npm run format
Cleaning and Building
Clean the project:


npm run clean
Build the project:


npm run build
